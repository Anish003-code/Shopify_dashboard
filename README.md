# 🛒 Shopify Sales & Customer Analytics Dashboard  

## 📰 Headline  
Analyzing e-commerce sales performance, customer behavior, product trends, and revenue insights across multiple product categories and regions.  

---

## 📌 Short Description  
An interactive and data-driven Power BI dashboard designed to analyze Shopify store operations and overall e-commerce performance. The dashboard provides insights into sales trends, customer purchasing behavior, product category performance, regional sales distribution, payment methods, and revenue generation to support business growth and data-driven decision-making.  

---

## ⚙️ Tech Stack  
- Power BI Desktop: Used for creating interactive dashboards and visualizations.  
- Power Query: Performed data cleaning, transformation, and preprocessing.  
- SQL: Used for querying and analyzing Shopify datasets.  
- Excel / CSV: Data source handling and preprocessing.  
- DAX (Data Analysis Expressions): Created KPIs, calculated measures, and dynamic business metrics.  
- GitHub: Version control and project documentation.  
- File Formats: `.pbix` for dashboard development and `.png` for dashboard preview.  

---

## 🔹 Business Problem  
E-commerce businesses process thousands of transactions daily, making it difficult to manually monitor sales performance, customer purchasing patterns, product demand, and operational efficiency. Businesses often struggle to identify top-performing products, understand customer behavior, and track revenue growth effectively.  

---

## 🔹 Goal of the Dashboard  
- Monitor total sales, revenue, and order performance  
- Analyze customer purchasing behavior and retention trends  
- Identify top-performing products and categories  
- Track regional and city-wise sales distribution  
- Measure average order value and transaction trends  
- Analyze payment method preferences  
- Support business decisions using interactive analytics  

---

## 📂 Project Workflow  
1. Collected and imported Shopify e-commerce datasets.  
2. Cleaned and transformed raw data using Power Query.  
3. Queried and analyzed datasets using SQL.  
4. Created data models and relationships in Power BI.  
5. Developed KPIs and calculated measures using DAX.  
6. Designed interactive dashboard visuals, slicers, and filters.  
7. Generated business insights and recommendations.  
8. Published the project on GitHub for portfolio showcase.
